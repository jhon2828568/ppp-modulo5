El login permite identificar a cada usuario antes de ingresar a la plataforma. De esta manera solo las personas autorizadas pueden acceder a determinadas funciones. Las rutas protegidas evitan que cualquier persona consulte o modifique información interna. También ayudan a mantener la seguridad del sistema. No deben exponerse contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
